# Anatomic Groupings Reference

## Overview

Anatomic groupings determine which repairs can be aggregated and which site-specific codes to use.

## Aggregation Rules Summary

### DO Aggregate (sum lengths, bill once):
- Same complexity (simple/intermediate/complex)
- Same anatomic group
- Same session

### DON'T Aggregate (bill separately):
- Different complexity levels
- Different anatomic groups
- Different sessions

---

## Site-Specific Value Hierarchy

**Higher-paying sites (in order):**

1. **Eyelids, nose, ears, lips** - Highest
2. **Face, forehead, cheeks** - High
3. **Scalp, neck, hands, feet, genitalia** - Medium
4. **Trunk, arms, legs** - Base

**Key:** Always document specific anatomic site to ensure highest appropriate code.
