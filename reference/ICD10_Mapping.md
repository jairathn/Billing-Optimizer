# ICD-10 Mapping for Dermatology

## Overview

Proper ICD-10 coding supports medical necessity for procedures. This reference maps common dermatology conditions to their ICD-10 codes.

## Procedure-Diagnosis Pairing Guidelines

### Destruction (17000-17004)
- Use L57.0 for AKs

### Destruction (17110-17111)
- Use B07.x for warts
- Use B08.1 for molluscum
- Use L82.x for SKs (with medical necessity)

### Biopsy
- Use suspected diagnosis or R22.x (localized swelling) if uncertain

### Excision - Benign
- Use D22.x, D23.x, L72.x, D17.x as appropriate

### Excision - Malignant
- Must have C44.x or C43.x confirmed by pathology

### IL Injection
- L91.0 for keloids
- L63.x for alopecia areata
- L70.x for acne cysts
- L40.x for psoriatic plaques
