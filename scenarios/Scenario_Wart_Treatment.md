# Clinical Scenario: Wart Treatment Visit

## Overview

Wart destruction is a common, billable procedure. Optimization requires counting lesions and using appropriate destruction vs injection codes.

---

## Destruction Codes (17110/17111)

| Lesion Count | Code | wRVU |
|--------------|------|------|
| 1-14 | 17110 | 0.52 |
| 15+ | 17111 | 0.79 |

## Chemotherapy Injection (96405/96406)

For intralesional 5-FU or bleomycin:

| Lesion Count | Code | wRVU |
|--------------|------|------|
| 1-7 | 96405 | 0.52 |
| 8+ | 96406 | 0.80 |

**Key:** Chemotherapy agents use 96405/96406, NOT 11900!

---

## Visit Optimization

### Billing Examples

**Cryo 8 warts:**
- 99213-25 + 17110 + G2211 = 1.30 + 0.52 + 0.33 = 2.15 wRVU

**Cryo 20 warts:**
- 99213-25 + 17111 + G2211 = 1.30 + 0.79 + 0.33 = 2.42 wRVU

**IL 5-FU 5 warts:**
- 99213-25 + 96405 + G2211 = 1.30 + 0.52 + 0.33 = 2.15 wRVU
