# Clinical Scenario: Alopecia

## Overview

Alopecia visits offer opportunities for IL injections, biopsies, and treatment of coexisting scalp conditions.

---

## Types and Opportunities

### Alopecia Areata

**IL Injection (11900/11901):**
- Inject patches with triamcinolone 5-10mg/mL
- Count each patch as lesion
- wRVU: 0.51 (≤7) / 0.78 (>7)

**Biopsy if diagnosis uncertain:**
- Punch (11104): 0.81 wRVU

**Comorbidities:**
- Nail pitting → document for complexity
- Atopy → document

### Androgenetic Alopecia

**Scalp biopsy if diagnosis uncertain:**
- Punch preferred for follicular evaluation

**Coexisting conditions:**
- Seborrheic dermatitis → separate diagnosis
- Telogen effluvium component

### Scarring Alopecia

**Biopsy essential:**
- Punch for adequate specimen
- Multiple may be needed

**Higher MDM:**
- Chronic, progressive condition
- Complex workup
