# Clinical Scenario: Full Body Skin Exam / Skin Check

## Overview

The routine skin check is a high-opportunity visit. Beyond the screening function, systematically assessing each body area can identify numerous billable conditions and procedures.

---

## Baseline Billing

### E/M Level Determination

| Scenario | E/M Level | MDM Justification |
|----------|-----------|-------------------|
| Screening, no concerning findings | 99213 | Low: Preventive assessment |
| History of skin cancer, surveillance | 99213-99214 | Low-Moderate: Monitoring known risk |
| Biopsy performed | 99213-25 | Separate from procedure |
| Multiple diagnoses/treatments | 99214 | Moderate: Multiple conditions |

### G2211 Applicability

- **Screening patient with no history:** Generally no (not chronic condition management)
- **Skin cancer surveillance:** YES - ongoing monitoring is chronic management
- **Patient with other chronic skin conditions:** YES

## The Systematic Body Sweep

### Scalp
**Look for:**
- Actinic keratoses → destruction (17000+)
- Seborrheic keratoses (symptomatic) → removal (17110)
- Suspicious lesions → biopsy (11102+)
- Seborrheic dermatitis → document, treat
- Alopecia → document, may warrant workup

### Face
**Look for:**
- AKs → destruction
- BCCs (pearly papules, telangiectasia)
- SCCs (scaly, indurated)
- Seborrheic keratoses → if symptomatic, remove
- Sebaceous hyperplasia
- Rosacea → document if present
- Lentigines → monitor vs biopsy

### Ears
**Look for:**
- AKs (very common, often missed)
- SCCs (high-risk site)
- BCCs
- Chondrodermatitis nodularis helicis

### Neck
**Look for:**
- Skin tags → removal if symptomatic (11200)
- AKs
- Poikiloderma of Civatte

### Trunk (Chest, Back, Abdomen)
**Look for:**
- Atypical nevi → biopsy if concerning
- Seborrheic keratoses → if patient wants removal
- Cherry angiomas (usually not treated)
- Melanoma (ABCDE criteria)
- Dermatofibromas (document)

### Arms
**Look for:**
- AKs (dorsal forearms especially)
- SKs
- Suspicious nevi
- Stucco keratoses

### Hands
**Look for:**
- AKs (dorsal hands)
- Warts → destruction (17110)
- Nail changes → debridement if needed
- Lentigines

### Legs
**Look for:**
- Stasis changes → may need workup
- Nevi
- Stucco keratoses
- Xerosis (opportunity for counseling)

### Feet
**Look for:**
- Warts (plantar) → destruction
- Tinea pedis → separate diagnosis/treatment
- Onychomycosis → nail debridement
- Melanoma (check between toes, soles)

### Nails (All)
**Look for:**
- Dystrophy → debridement (11720/11721)
- Melanonychia → biopsy (11755)
- Psoriatic changes
- Onychomycosis

### Intertriginous Areas
**Look for:**
- Intertrigo → treatment
- Tinea → treatment
- Inverse psoriasis
- Skin tags → removal

## Procedure Opportunities

### Biopsy (11102-11107)

**Trigger:** Any concerning lesion
- Changing mole
- New lesion in older patient
- Irregular pigment pattern
- Patient concern about lesion

**Coding:**
- First biopsy: 11102 (shave), 11104 (punch), 11106 (incisional)
- Additional: 11103, 11105, 11107

### Destruction - Premalignant (17000-17004)

**Trigger:** Actinic keratoses
- Count EVERY lesion
- Bill 17000 (first) + 17003 (each 2-14) OR 17004 (15+)

### Destruction - Benign (17110/17111)

**Trigger:** Warts, molluscum, symptomatic SKs
- Medical indication required (symptoms, functional, diagnostic uncertainty)

### Skin Tag Removal (11200/11201)

**Trigger:** Symptomatic skin tags
- Up to 15 lesions: 11200
- Each additional 10: 11201

### Shave Removal (11300-11313)

**Trigger:** Symptomatic benign lesions requiring removal
- Site-specific coding (face > trunk)
- Measure lesion size

### Nail Debridement (11720/11721)

**Trigger:** Dystrophic nails found incidentally
- 1-5 nails: 11720
- 6+ nails: 11721

## Billing Examples

### Example 1: Routine Skin Check, No Findings

**Visit:** Annual skin check, no concerning lesions

**Billing:**
- 99213

**wRVU:** 1.30

### Example 2: Skin Check with AK Destruction

**Visit:** Found and treated 8 AKs

**Billing:**
- 99213-25
- 17000 + 17003 x7

**wRVU:** 1.30 + 0.61 + 0.63 = 2.54

### Example 3: Skin Check with Biopsy

**Visit:** Concerning lesion biopsied

**Billing:**
- 99213-25
- 11102 (or 11104)

**wRVU:** 1.30 + 0.64 (or 0.81) = 1.94-2.11

### Example 4: Skin Check with Multiple Findings

**Visit:** 5 AKs destroyed, 2 biopsies, skin tags removed

**Billing:**
- 99214-25 (multiple conditions)
- 17000 + 17003 x4 (AKs)
- 11102 + 11103 (biopsies)
- 11200 (skin tags)

**wRVU:** 1.92 + 0.97 + 1.01 + 0.80 = 4.70

### Example 5: Skin Cancer Surveillance

**Visit:** History of melanoma, surveillance exam

**Billing:**
- 99213 (or 99214 if multiple sites/concerns)
- G2211 (ongoing surveillance = chronic management)
- Any procedures performed

**wRVU:** 1.30 + 0.33 = 1.63 (plus procedures)

## "Next Time" Teaching Points

### If AKs not counted:
> "Count and document every AK treated. Each additional lesion adds 0.09 wRVU (17003)."

### If warts found but not treated:
> "Wart destruction (17110) is billable. Ask if lesions are bothersome."

### If dystrophic nails not addressed:
> "Check nails at every skin exam. Dystrophic nails can be debrided (11720/11721)."

### If symptomatic SKs not offered removal:
> "Ask patients if any growths are symptomatic. SK removal is billable with medical indication."

### If skin check without surveillance history:
> "For skin cancer surveillance patients, add G2211 as ongoing chronic management."
