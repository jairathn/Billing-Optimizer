# Clinical Scenario: Phototherapy

## Overview

Phototherapy provides ongoing per-session billing for patients with psoriasis, vitiligo, eczema, and other photosensitive conditions.

---

## Codes

| Code | Description | wRVU |
|------|-------------|------|
| 96900 | Actinotherapy (UV light) | 0.26 |
| 96910 | Photochemotherapy (tar/petrolatum + UV) | 0.41 |
| 96912 | PUVA | 0.37 |
| 96913 | Photochemotherapy (extracorporeal) | 2.82 |

---

## Indications

### 96900 - NB-UVB
- Psoriasis
- Vitiligo
- Atopic dermatitis
- Mycosis fungoides
- Pruritus

### 96912 - PUVA
- Psoriasis (severe/recalcitrant)
- Vitiligo
- Mycosis fungoides
- Other indications requiring psoralen

---

## Billing Structure

**Each session is separately billable.**

Typical course: 2-3x weekly for 10-30+ sessions

**Revenue potential:** 30 sessions × 0.26 = 7.8 wRVU (NB-UVB)

---

## E/M + Phototherapy

**Initiation visit:**
- 99214 + G2211 (disease evaluation, treatment planning)

**Monitoring visits:**
- Periodic E/M (monthly or as needed)
- Document response, adjust dosing

---

## Documentation

### Each Session
> "Phototherapy session #[X]. Diagnosis: [psoriasis/vitiligo/etc]. Treatment: [NB-UVB/PUVA]. Dose: [X] mJ/cm². Body areas treated: [list]. Patient tolerated well. No adverse effects."

### Monitoring Visit
> "Follow-up for [condition] on phototherapy. Currently session #[X] of [Y] planned. [X]% improvement noted. Current dose: [X] mJ/cm². Continue protocol. Next assessment in [timeframe]."

---

## Optimization

- Bill each session
- Add G2211 to E/M visits for chronic management
- Document response for medical necessity
- Track cumulative dose for safety
