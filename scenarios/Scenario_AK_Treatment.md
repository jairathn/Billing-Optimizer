# Clinical Scenario: AK Treatment Visit

## Overview
AK destruction is high-frequency billing. Key optimization: COUNT EVERY LESION.

---

## Destruction Codes

| Lesion Count | Code | wRVU |
|--------------|------|------|
| 1 | 17000 | 0.61 |
| 2 | 17000 + 17003 | 0.70 |
| 5 | 17000 + 17003×4 | 0.97 |
| 10 | 17000 + 17003×9 | 1.42 |
| 14 | 17000 + 17003×13 | 1.78 |
| 15+ | 17004 | 2.59 |

**Critical:** If 15 or more AKs, use 17004 alone (better value).

---

## Visit Optimization

### Count Every Lesion
- Each physically separate AK = 1 lesion
- Document location and count per site

### Check All Sun-Damaged Areas
- Scalp (especially bald)
- Face
- Ears (commonly missed!)
- Neck
- Dorsal hands
- Forearms

---

## Billing Examples

### 8 AKs Treated
- 99213-25 + 17000 + 17003×7 = 1.30 + 0.61 + 0.63 = 2.54 wRVU

### 20 AKs Treated
- 99213-25 + 17004 = 1.30 + 2.59 = 3.89 wRVU
