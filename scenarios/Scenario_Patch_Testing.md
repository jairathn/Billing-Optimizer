# Clinical Scenario: Patch Testing

## Overview

Patch testing is a high-value service for contact dermatitis workup with per-allergen billing.

---

## Codes

| Code | Description | Unit |
|------|-------------|------|
| 95044 | Patch test | Per allergen |
| 95052 | Photo patch test | Per allergen |
| 95060 | Ophthalmic mucous membrane | Each substance |

---

## Standard Panels

### Standard Series (80+ allergens)
Each allergen billed separately with 95044

### Extended Panels
- Additional specific allergen series
- Fragrance, cosmetic, hairdresser, etc.

### Photo Patch
- For suspected photoallergic contact dermatitis
- 95052 for UV-exposed allergens

---

## Visit Structure

### Visit 1: Application
- E/M (99214-25) + patches applied
- Document number of allergens tested

### Visit 2: Reading (Day 2)
- E/M (99212/99213)
- Document reactions

### Visit 3: Delayed Reading (Day 4-5)
- E/M (99213/99214)
- Final interpretation
- Patient education

## Documentation

### Application Visit
> "Patch testing initiated for suspected allergic contact dermatitis. [Number] allergens applied to upper back. Standard [series name] panel plus [additional panels]. Return in 48 hours for reading. Instructed to keep patches dry."

### Reading Visit
> "Patch test reading at [48/96] hours. Results: [List positive reactions with grade (+, ++, +++)]. [List negative/irritant reactions]. Discussion of relevance to patient's dermatitis. Avoidance counseling provided."

## "Next Time" Points

- Bill each allergen separately (95044)
- Consider photo patch (95052) for photodistributed dermatitis
- E/M billable at each reading visit
- G2211 applicable if contact dermatitis is chronic
