# Clinical Scenario: Eczema / Atopic Dermatitis

## Overview

Eczema is a chronic condition with significant billing opportunities through proper MDM documentation, comorbidity identification, and procedure capture.

---

## Baseline Billing

### E/M Level

| Scenario | E/M | Justification |
|----------|-----|---------------|
| Stable on topicals | 99213 | Chronic stable, topical Rx |
| Flare, adjusting Rx | 99214 | Mild exacerbation |
| Dupixent/biologic | 99214 | Drug requiring monitoring |
| Severe/erythrodermic | 99215 | Threat to function |

### G2211: Always Add

Chronic condition requiring ongoing management.

---

## Procedure Opportunities

### IL Injection for Lichenification (11900)
- Thick, lichenified plaques
- wRVU: 0.51

### Patch Testing Referral/Performance (95044)
- If contact component suspected
- Per allergen tested

---

## Comorbidities to Look For

| Finding | Opportunity |
|---------|-------------|
| Secondary impetiginization | Separate bacterial diagnosis, higher MDM |
| Eczema herpeticum | High urgency, 99215 justified |
| Lichenification | IL injection |
| Contact component | Patch testing |
| Asthma/allergic rhinitis | Document atopic triad |

---

## Billing Examples

### Standard Follow-up
- 99213 + G2211 = 1.63 wRVU

### Flare with Lichenification Injection
- 99214-25 + 11900 + G2211 = 2.76 wRVU

### Dupixent Management
- 99214 + G2211 = 2.25 wRVU

---

## Documentation Template

> "Follow-up for atopic dermatitis. Current treatments: [list]. BSA: [X]%. Locations: [list]. [Lichenification/excoriation/xerosis] noted. Secondary infection: [yes/no]. IGA score: [0-4]. Continue [regimen]. Return [timeframe]."

---

## "Next Time" Points

- Look for lichenified areas → IL injection (0.51 wRVU)
- Screen for contact component → patch testing
- Document secondary infection if present → higher MDM
- Always add G2211 for chronic management
